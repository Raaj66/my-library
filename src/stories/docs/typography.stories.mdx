# Typography

iCapital Network platform uses one overarching typeface: Avenir Next LT Pro. The fallback font stack is: Helvetica,Arial,sans-serif

The platform uses four main font weights of Avenir Next LT Pro:

- Avenir Next LT Pro Medium
- Avenir Next LT Pro Demi
- Avenir Next LT Pro Light
- Avenir Next LT Pro Bold
